What is Bootstrap?

Bootstrap is a free and open source of code written in HTML, CSS, and JavaScript.It is a CSS framework for developing responsive and mobile-first websites.

History

Bootstrap is originally created by a small team of developers at Twitter on GitHub.It was created in mid-2010 at Twitter and was know at that time as Twitter Blueprint.It was originally released on August 2011, and kept on updating through different vesrions with better features.

Colors

In bootstrap code, to color the text, you just have to add a class of different colors that has specific names such as; the blue color's name in bootstrap is called primary so just add class"text-primary" and the text will be blue, the grey color is called secondary so you add class"text-secondary", the black color is called dark so you add class"text-dark" and so on....

.text-danger

Utilities

Spacing

In bootstrap margins and paddings are used in classes to fix any text or images positions. Margin is "m" and padding is "p".
In case of giving a space at the top it's always "t" and the bottom "b".
Adding the distance by numbers such as;

  1. "1" sets margin or padding to 0.25rem (4px if font-size is 16px).
  2. "2" sets margin or padding to 0.5rem (8px if font-size is 16px)
...and so on going with the calculations.

For example: margin-top with distance added in any place, just add to any class mt-4 or mt-5 or any other chosen number depending on the distance needed.
